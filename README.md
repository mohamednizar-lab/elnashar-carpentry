# El Sayed El Nashar Carpentry Works

السيد النشار لأعمال النجارة - Production-ready carpentry business website.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy (Vercel)

Project includes `vercel.json`. Connect repo to Vercel or run:

```bash
npx vercel
```
